Para la ejecución de modelo únicamente hay que seguir los siguientes pasos:
1. Ejecutar Test1.py para crear la bolsa de palabras y generar el fichero a evaluar.
2. Ejecutar Test2.py para obtener los tiempos de ejecución por la aplicación de varios modelos.

Por defecto el fichero Test1.py trabajara con una bolsa de 1000 palabras.
